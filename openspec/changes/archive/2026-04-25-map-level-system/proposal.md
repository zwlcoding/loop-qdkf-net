# Proposal: Map Level System

## Problem
当前地图类型单一，缺乏地形变化和战术深度，重玩性不足。

## Solution
实现地图关卡系统：多种地形类型、动态地图布局、关卡解锁机制，增加战术选择和重玩性。

## Scope
- 地形类型系统（平原、山地、城市、森林等）
- 地图布局生成（预设 + 随机元素）
- 地形效果（移动消耗、掩体、视野遮挡）
- 关卡解锁机制
- 地图预览界面

## Out of Scope
- 完全程序化生成
- 地图编辑器
- 多层地图

## Success Criteria
1. 至少 4 种不同地形类型，各有独特效果
2. 地图布局有变化，不是固定模板
3. 地形效果影响战斗（移动、掩体、视野）
4. 关卡按进度解锁
5. 手机端地图清晰可读
