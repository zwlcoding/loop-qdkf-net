# Proposal: Mission Extraction Loop

## Problem
战斗系统已完成，但没有 roguelite 核心循环。玩家打赢战斗后没有目标感，无法推进游戏进度。

## Solution
实现完整的任务提取循环：选择任务 → 进入战斗 → 战斗中可提取 → 提取判定 → 奖励结算 → 进度保存 → 下一轮准备。

## Scope
- 任务选择界面（战前）
- 提取机制（战斗中触发）
- 提取判定逻辑（成功/失败）
- 奖励结算（资源、经验、解锁）
- 进度保存/读取
- 下一轮准备界面

## Out of Scope
- 多结局系统
- 成就系统
- 排行榜

## Success Criteria
1. 玩家可以从任务列表选择任务进入战斗
2. 战斗中满足条件后可以触发提取
3. 提取成功后获得奖励并保存进度
4. 进度可以在下次启动时恢复
5. 手机端完整可用
