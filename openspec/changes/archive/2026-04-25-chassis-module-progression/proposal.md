# Proposal: Chassis Module Progression

## Problem
战斗前没有装备/loadout 系统，玩家无法自定义队伍配置，缺乏战前决策空间。

## Solution
实现底盘模块进度系统：玩家可以为每个单位选择底盘类型、装备模块、配置技能，形成不同的战斗风格。

## Scope
- 底盘类型系统（不同底盘有不同属性）
- 模块装备系统（武器、护甲、技能模块）
- Loadout 配置界面
- 战斗中底盘/模块效果生效
- 奖励解锁新底盘/模块

## Out of Scope
- 模块升级/强化系统
- 模块合成系统
- 模块交易系统

## Success Criteria
1. 玩家可以在战前为每个单位选择底盘类型
2. 玩家可以为单位装备不同模块
3. 底盘和模块效果在战斗中生效
4. 通过任务奖励解锁新底盘/模块
5. 手机端完整可用
